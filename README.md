# Práctica GitHub DAW

Repositorio de prueba para aprender Git y GitHub en Debian.

Autor: Francisco González Fecha: 9-2-2026

¿Qué es GitHub?: Es una plataforma de alojamiento en la nube para proyectos que utilizan Git. Permite la colaboración y el control de versiones remoto.

Tipos de repositorios: Públicos (todo el mundo los ve) y Privados (solo tú y tus colaboradores).
